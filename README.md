# 🏥 Health Care Portal

The Health Care Portal is a web-based application developed using Django. It allows patients to book appointments, view doctor information, and manage healthcare services through a simple and user-friendly interface.

---

## ✨ Features

- 👨‍⚕️ Doctor Information
- 📅 Appointment Booking
- 📝 Update Appointments
- 📧 Patient Email Registration
- 📱 Responsive User Interface
- 🔐 Django Admin Panel

---

## 🛠 Technologies Used

- Python
- Django
- HTML5
- CSS3
- Bootstrap
- SQLite3
